ml4pers: Machine Learning for Persistence

You’re on the associated page for our repository for our paper, A Comparative Study of Machine Learning Methods for Persistence Diagrams.

Soon this code will be available as a python package, with additional documentation and examples. Note there are many packages that have similar or the same types of functions - the goal of this package is to provide datasets and precomputed persistence diagrams with some examples of how to build machine learning models from the persistence diagrams. This may at some point be merged with another package.
